nodeblast
=========
a node.js wrapper for bl2seq from BLAST (not BLAST+) so it can be offloaded to another server

In theory this should be faster than a CGI wrapper.

I use this for aligning stuff to the mitochondrial genome.



Usage:
`node nodeblast.js`
